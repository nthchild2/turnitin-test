# Artifact Rules

Files in `docs/run-ledger.md` and `templates/` are framework artifacts.

- Do not modify templates unless explicitly asked.
- Run ledger entries are append-only. Do not edit or delete past entries.
- Use the exact format defined in each template. Do not improvise structure.
- Every completed pipeline run must append a new entry to the run ledger.
